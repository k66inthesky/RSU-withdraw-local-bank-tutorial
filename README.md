### 常見問題
+ 我幫朋友詢問是否可以不與美林聯繫直接售出股票後匯款至台灣的銀行，與美林聯繫得到結果是No!必須先和美林聯繫。
+ Tax Certification填地址記得別填錯! 否則會多卡約兩個工作天。
+ Tax Certification表格的FTIN外國人稅籍編號: 台灣人填身分證字號。
+ Wire Instructions填各家銀行的Swift Code上網找就有，格式是8碼英文+後3碼數字(但富邦客服教我其實填前8碼就可)。
